# Onboarding Documentation

This folder contains onboarding guides and resources for new team members joining biased-cms.

## What to Include

### General Onboarding
- Welcome and company overview
- Culture and values
- Team structure
- Communication channels
- Tools and access

### Role-Specific Onboarding
- Developer onboarding
- Designer onboarding
- Product manager onboarding
- Sales/Marketing onboarding
- Support onboarding

### First Week/Month Checklists
- Day 1 checklist
- Week 1 goals
- 30-60-90 day plans
- Key contacts

### Resources
- Glossary of terms
- FAQ
- Who to ask for what
- Learning resources
