## 1.2 Logo Usage & Rules

The logo set consists of a Primary Logo (text + icon) and a standalone Icon/Favicon.

### Primary Logo (Logo.png)

* **Usage:** Prominent placement in the main website header.
* **Purpose:** Full brand identification.

### Icon/Favicon (logo2.jpg)

* **Usage:** Favicon, social media profiles, small branding spots, and loading screens.
* **Purpose:** Compact brand recognition.

### Clear Space Rule

* **Rule:** Maintain clear space around the logo equal to at least the **height of the "B"** in "BIASED."
* **Minimum Size:** Must ensure all text and inner circle details are clearly legible.

## 1.3 Accessibility Guidelines

| Requirement | Detail |
| :--- | :--- |
| **Contrast** | Ensure sufficient color contrast for all text and UI elements against backgrounds. |
| **Semantics** | Use semantic HTML for structure and meaning. |
| **Keyboard Nav** | Ensure full and clear keyboard navigation functionality. |
| **Imagery** | Provide descriptive alt text for all meaningful images and graphics. |
