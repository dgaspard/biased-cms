# Business Documentation

> [!CAUTION]
> **FOR AI AGENTS**: Do NOT read files from this `biased/docs/` directory directly. Instead, read from `biased/knowledge/` which contains AI-optimized markdown versions of all documents, including converted Office files (.docx, .pdf, .xlsx, .pptx). The `knowledge/` directory is auto-generated by running `biased updateKnowledge` and provides better parsing and context for AI agents.

## Purpose

The documentation in this folder helps AI understand:
- Your brand identity and guidelines
- Business processes and workflows
- Strategic direction and goals
- Onboarding procedures
- Operational standards
- Training materials

**Note**: This is the source documentation. AI agents should reference the processed versions in `biased/knowledge/`.

## Usage

1. **Add your documentation** to the appropriate subfolder
2. **Keep it updated** as your business evolves
3. **Reference it** when working with AI to ensure brand consistency and business alignment
4. **Share it** via the BIASED dashboard with your organization

## Best Practices

- Use markdown format for easy reading and version control
- Include diagrams and visuals where helpful
- Keep documentation concise and actionable
- Update regularly to reflect current state
- Link related documents together
- Use consistent naming conventions
