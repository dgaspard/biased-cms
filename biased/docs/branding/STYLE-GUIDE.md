## 2.1 Color Palette

All color definitions are in HEX format.

### Primary Brand Colors

| Name | HEX Code | Purpose | Representation |

| **Dark Green** | `#3B7B4B` | Primary action, reliability, and brand accent. | Reliability, Growth, Accuracy |

| **Dark Teal** | `#3F6E7B` | Data depth, stable elements, integrity. | Depth, Data, Integrity |

### Text & Background Colors

| Name | HEX Code | Purpose |
| :--- | :--- | :--- |

| **Secondary Text**| `#5A5A5A` | Subheadings, descriptive text, captions. |

## 2.2 Typography

The typeface is a clean, modern, readable sans-serif.

### Typefaces

* **Primary Font (Headings):** **Open Sans Bold** or **Montserrat Bold** (if Open Sans is unavailable).
* **Body Font (Paragraphs, UI):** **Open Sans Regular** or **Lato Regular**.

### Responsive Font Sizes (Base: Desktop/Mobile)

| Element | Desktop Size (px) | Mobile Size (px) | Weight | Color |
| :--- | :--- | :--- | :--- | :--- |
| **H1 (Page Title)**| 36px - 48px | 28px - 36px | Bold | Main Text (`#2D2D2D`) |
| **H2 (Section Title)**| 28px - 36px | 22px - 28px | Bold | Main Text (`#2D2D2D`) / Dark Teal (`#3F6E7B`) |
| **H3 (Sub-section)**| 20px - 24px | 18px - 22px | Bold/Semi-Bold| Secondary Text (`#5A5A5A`) |
| **Body Text** | 16px - 18px | 14px - 16px | Regular | Main Text (`#2D2D2D`) |
| **Small Text** | 12px - 14px | 12px | Regular | Secondary Text (`#5A5A5A`) |

## 2.3 Imagery & Graphics

* **Style:** Clean, data-focused, and conceptual. Avoid busy or complex photographic textures.
* **Themes:** Data visualization, analytical processes, objectivity, transparency.
* **Illustration:** Simple, flat, or subtly textured illustrations only, using the defined color palette.
* **Data Visualization:** All charts, graphs, and data elements **must** use the primary brand colors.
